## 外テスト用チェックリスト

### 1. 持ち出す前の確認

* ESP32に最新のv0.3コードを書き込んだ
* モバイルバッテリーでESP32が起動する
* 起動時の振動合図が分かる
* キャリブレーション完了時の振動合図が分かる
* 地磁気センサー基板の「前」印が分かる
* 配線が抜けにくい状態になっている
* モーター、地磁気センサー、ESP32、モバイルバッテリーの位置関係を記録した

### 2. テスト場所

* 金属製の柵や手すりから少し離れた場所で試す
* 自転車、車、金属製ベンチの近くを避ける
* スマートフォンやモバイルバッテリーを地磁気センサーに近づけすぎない
* まずは広めで安全に立ち止まれる場所で試す
* 歩きながら試す前に、立ち止まった状態で確認する

### 3. 起動・キャリブレーション

* モバイルバッテリーに接続して起動する
* 起動合図の振動を確認する
* 10秒間、地磁気センサーを水平にしてゆっくり一周以上回す
* キャリブレーション完了の振動を確認する
* 反応がおかしい場合は、ESP32のENボタンを押してやり直す

### 4. 北検知の確認

* 「前」印を北方向に向けたときに振動する
* 北から少し外すと断続振動になる
* 北から大きく外すと振動が止まる
* もう一度北に戻すと再び振動する
* 真北付近で振動が不規則にならない
* 北に近づくほど振動間隔が短くなる感覚がある
* 真北付近で連続振動する

### 5. 環境による変化の確認

* 場所を少し変えても北検知できる
* 建物の近くと離れた場所で反応に違いがあるか確認する
* 金属物の近くで反応が乱れるか確認する
* スマートフォンを近づけたときに反応が変わるか確認する
* モバイルバッテリーの位置を変えると反応が変わるか確認する
* モーターと地磁気センサーの距離で反応が変わるか確認する

### 6. 持ちやすさ・体験の確認

* 片手で持ちやすい
* 地磁気センサーを水平に保ちやすい
* 「前」印の方向が直感的に分かる
* 振動の強さが分かりやすい
* 振動の間隔変化で北に近づいている感覚がある
* 画面を見なくても状態が分かる
* 歩きながら使えそうか確認する

### 7. テスト後に記録すること

* テスト日
* テスト場所
* 天気や周囲の環境
* モバイルバッテリーで正常に起動したか
* キャリブレーションがうまくいったか
* 北検知の反応は自然だったか
* 振動の強さは十分だったか
* 誤反応が起きた場所や状況
* 配線や持ち運びで不安だった点
* 次に直したいこと

### 8. 次回改善候補

* センサーとモーターの距離を離す
* センサーとモバイルバッテリーの距離を離す
* 配線を固定する
* ブレッドボードを使わない構成を検討する
* 簡易ケースを作る
* キャリブレーション操作をもっと分かりやすくする
* 振動パターンをさらに調整する
* 屋外で使いやすい持ち方を決める
