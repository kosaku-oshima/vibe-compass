# vibe-compass

## 概要

ESP32と地磁気センサーを使い、北を向いたときに振動モーターが動作する試作品です。

スマートフォンの画面を見続けるのではなく、方角を「振動で感じる」ことで、自分の感覚を使いながら目的地へ向かう体験を目指しています。

まずは、地磁気センサーが北を検知したときに振動するシンプルなデバイスを試作しています。

## コンセプト

振動コンパスは、方角を画面で確認するのではなく、身体感覚として受け取るための試作ガジェットです。

現在の試作では、地磁気センサー基板の「前」と決めた方向が北を向いたときに振動モーターが動作します。北に近づくほど振動の間隔が短くなり、真北付近では連続振動するようにしています。

## 現在の状態

* ESP32と地磁気センサーを接続済み
* 地磁気センサーから方位を取得
* 地磁気センサーの値をもとに北方向を判定
* 北に近づくほど振動間隔が短くなる
* 真北付近では連続振動する
* モバイルバッテリー給電でPCなし動作が可能
* 起動時とキャリブレーション完了時に振動で状態を通知

## 使い方

1. ESP32をモバイルバッテリー、またはPCに接続する
2. 起動後、短い振動で起動を確認する
3. 起動直後の10秒間、地磁気センサーを水平にしてゆっくり一周以上回す
4. キャリブレーション完了後、長めの振動で完了を確認する
5. 地磁気センサー基板の「前」印を北に向けると振動する

反応がおかしい場合は、ESP32のENボタンを押してリセットし、再度キャリブレーションを行います。

## キャリブレーションについて

起動直後に10秒間、地磁気センサーを水平にして回転させます。

これは、地磁気センサーのX/Y方向の中心ズレを補正するための処理です。PCや外部コンパスがなくても使えるようにするため、起動時に自動でキャリブレーションを行う仕様にしています。

## バージョン

### v0.1：北検知＋振動成功

* ESP32と地磁気センサーを接続
* 地磁気センサーから方位を取得
* Serial Monitorで `n` を入力した時点の方位を北として記録
* 記録した北方向を向いたときに振動モーターが動作

### v0.2：北への近さを振動で表現

* 北からのズレ角度を計算
* ズレ角度に応じて振動パターンを変化
* 北に近づくほど振動間隔が短くなる体験を検証
* 真北付近では連続振動するように調整

### v0.3：自律北検知版

* Serial Monitorで `n` を入力して北を登録する方式を廃止
* 地磁気センサー自身の方位計算を使って北を判定
* `SENSOR_FORWARD_OFFSET` により、基板の「前」方向とセンサー値のズレを補正
* PCや外部コンパスなしで動作可能
* 起動時とキャリブレーション完了時に振動で状態を通知
* 不規則だった断続振動を改善し、北に近づくほど振動間隔が短くなるように修正

## 現在の設定メモ

* 振動モーター：GPIO25
* I2C SDA：GPIO21
* I2C SCL：GPIO22
* 北判定範囲：±25度
* 真北付近の連続振動範囲：おおよそ±5度
* `SENSOR_FORWARD_OFFSET`：99.3度

`SENSOR_FORWARD_OFFSET` は、地磁気センサー基板の「前」方向を北に向けたときのHeading値をもとに設定しています。

## 注意点

地磁気センサーは、周囲の金属や磁石、モーター、モバイルバッテリー、スマートフォンなどの影響を受ける可能性があります。

屋外テストでは、まず金属製の柵、自転車、車、ベンチ、建物の鉄骨などから少し離れた場所で確認するのがよさそうです。

また、現在の試作では地磁気センサーをなるべく水平に持つことを前提にしています。



## 外テスト用チェックリスト

### 1. 持ち出す前の確認

* ESP32に最新のv0.3コードを書き込んだ
* モバイルバッテリーでESP32が起動する
* 起動時の振動合図が分かる
* キャリブレーション完了時の振動合図が分かる
* 地磁気センサー基板の「前」印が分かる
* 配線が抜けにくい状態になっている
* モーター、地磁気センサー、ESP32、モバイルバッテリーの位置関係を記録した

### 2. テスト場所

* 金属製の柵や手すりから少し離れた場所で試す
* 自転車、車、金属製ベンチの近くを避ける
* スマートフォンやモバイルバッテリーを地磁気センサーに近づけすぎない
* まずは広めで安全に立ち止まれる場所で試す
* 歩きながら試す前に、立ち止まった状態で確認する

### 3. 起動・キャリブレーション

* モバイルバッテリーに接続して起動する
* 起動合図の振動を確認する
* 10秒間、地磁気センサーを水平にしてゆっくり一周以上回す
* キャリブレーション完了の振動を確認する
* 反応がおかしい場合は、ESP32のENボタンを押してやり直す

### 4. 北検知の確認

* 「前」印を北方向に向けたときに振動する
* 北から少し外すと断続振動になる
* 北から大きく外すと振動が止まる
* もう一度北に戻すと再び振動する
* 真北付近で振動が不規則にならない
* 北に近づくほど振動間隔が短くなる感覚がある
* 真北付近で連続振動する

### 5. 環境による変化の確認

* 場所を少し変えても北検知できる
* 建物の近くと離れた場所で反応に違いがあるか確認する
* 金属物の近くで反応が乱れるか確認する
* スマートフォンを近づけたときに反応が変わるか確認する
* モバイルバッテリーの位置を変えると反応が変わるか確認する
* モーターと地磁気センサーの距離で反応が変わるか確認する
