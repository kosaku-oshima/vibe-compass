# 配線メモ v0.1

## 使用部品
- ESP32 DevKitC
- 地磁気センサー
- 振動モーター
- ブレッドボード
- ジャンパーワイヤ

## 接続
| 部品 | ピン | ESP32側 |
|---|---|---|
| 地磁気センサー | VIN | 3V3 |
| 地磁気センサー | GND | GND |
| 地磁気センサー | SDA | GPIO21 |
| 地磁気センサー | SCL | GPIO22 |
| 振動モーター | 制御 | GPIO21,22 |
| 振動モーター | GND | GND |

## メモ
北を向いたときにモーターが振動するところまで確認済み。
